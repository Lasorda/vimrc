## Quick Start

Install [nodejs](https://nodejs.org/en/download/) >= 10.12:

```sh
curl -sL install-node.now.sh/lts | bash
```

Download vimrc
```sh
curl -sL -o ~/.vimrc https://raw.githubusercontent.com/lasorda/vimrc/master/vimrc
```

## Features

* `ctrl ]` go to define
* `space o` outline display
* `space f` show files
